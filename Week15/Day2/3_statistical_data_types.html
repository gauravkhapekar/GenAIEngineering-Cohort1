Statistical Data Types

Understanding data types is fundamental to statistical analysis. Data can be classified into different categories based on their nature and characteristics.
Q
Qualitative Data (Categorical)

Data that describes qualities or characteristics that cannot be measured numerically.

Nominal Data

Categories with no inherent order or ranking. Each category is simply different from the others.

Examples:

Colors Gender Blood Type Country Eye Color Marital Status
Key Feature: No mathematical operations can be performed. Only counting and percentages make sense.

Ordinal Data

Categories that have a natural order or ranking, but the intervals between categories are not necessarily equal.

Examples:

Education Level Star Ratings Pain Scale Economic Status Survey Responses Military Rank
Key Feature: Order matters, but differences between ranks may not be equal.
Q
Quantitative Data (Numerical)

Data that can be measured and expressed numerically. Mathematical operations are meaningful.

Discrete Data

Countable data with distinct, separate values. Usually whole numbers with gaps between possible values.

Examples:

Number of Children Cars Sold Test Scores Population Count Books Read Website Clicks
Key Feature: Can be counted but not measured infinitely precisely.

Continuous Data

Measurable data that can take any value within a range. Can be measured to any degree of precision.

Examples:

Height Weight Temperature Time Distance Income
Key Feature: Can be measured infinitely precisely (limited only by measuring instruments).
